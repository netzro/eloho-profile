# Eloho Profile

Autonomous NGX Investment Agent for Hermes Agent.

## Quick Start

1. Install Hermes Agent

2. Clone this profile to ~/.hermes/profiles/eloho/

3. Run the gateway (from a separate shell, not inside gateway)

4. DM Eloho on Telegram

## Commands

- "Check prices" - Live NGX prices

- "Run DCA" - Execute DCA round

- "Give me a report" - Portfolio with PnL

- "Invoice" - Stripe invoice

- "Status" - System health
